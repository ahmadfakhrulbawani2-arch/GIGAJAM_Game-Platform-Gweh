# What the Heck is This?

So about a week ago, I attend my college laboratory event of game dev bootcamp called GIGAJAM that teaches me how to make a simple 2d game using Construct 3. For your information, construct 3 is a web app sass game dev engine that runs web based game that is using Javascripts/Typescripts with UI of HTML/CSS but made more easier of its event sheet based programming so that we don't need to write a lot of Js/Ts code. Here is a preview of this repo game:

  <img src="./docs/preview.gif" alt="preview">
</p>

Development Technologies:

1. Graphics from itch.io Pixel Adventure
2. Game SFX from Leshy SFMaker
3. Game engine with Construct 3

# How to run?

<details>
<summary>Requirements</summary>

- A web browser
- Live Server Extension or other web app runner

</details>

<details>
<summary>Step by step</summary>

- Create and open new empty folder
- In your empty folder clone this repository to your local computer
  ```bash
  git clone https://github.com/ahmadfakhrulbawani2-arch/GIGAJAM_Game-Platform-Gweh.git .
  ```
- Open index.html in root directory
- Open live server
- Done you can now play the game

</details>
